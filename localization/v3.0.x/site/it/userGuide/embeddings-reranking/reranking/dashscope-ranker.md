<ul>
<li><p>Una collezione Milvus con un campo " <code translate="no">VARCHAR</code> " contenente il testo da riorganizzare.</p></li>
<li><p>Una chiave API DashScope valida.</p></li>
<li><p>Accesso a un modello di riclassificazione DashScope, come <code translate="no">gte-rerank-v2</code>.</p></li>
</ul>
<p>Per i modelli di riclassificazione disponibili e gli endpoint regionali, consultare <a href="https://www.alibabacloud.com/help/en/model-studio/text-rerank-api">l’API Text Rerank</a> di <a href="https://www.alibabacloud.com/help/en/model-studio/text-rerank-api">Alibaba Cloud Model Studio</a>.</p>
