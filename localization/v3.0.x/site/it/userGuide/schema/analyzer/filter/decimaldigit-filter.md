<p>Per il testo arabo, l’analizzatore integrato <a href="/docs/it/arabic-analyzer.md"><code translate="no">arabic</code></a> include già il filtro " <code translate="no">decimaldigit</code> ". Utilizzare direttamente " <code translate="no">decimaldigit</code> " quando è necessaria la normalizzazione delle cifre in una pipeline di analizzatori personalizzata.</p>
</div>
<p>Per utilizzare il filtro " <code translate="no">decimaldigit</code> " in un analizzatore personalizzato, aggiungerlo alla sezione " <code translate="no">filter</code> " in <code translate="no">analyzer_params</code>:</p>
<pre><code translate="no" class="language-python">analyzer_params = {
    <span class="hljs-string">&quot;tokenizer&quot;</span>: <span class="hljs-string">&quot;standard&quot;</span>,
    <span class="hljs-string">&quot;filter&quot;</span>: [<span class="hljs-string">&quot;decimaldigit&quot;</span>],
}
<button class="copy-code-btn"></button></code></pre>
<p>Il filtro " <code translate="no">decimaldigit</code> " non presenta parametri configurabili.</p>
<p>Il filtro converte le cifre decimali Unicode, comprese quelle arabo-indiane, thailandesi, devanagari, bengalesi e a larghezza intera, in cifre ASCII. Opera sui token generati dal tokenizer. La configurazione sopra riportata è volutamente un esempio di analizzatore personalizzato e non include la pipeline completa di elaborazione araba.</p>
