<p>스냅샷을 만드는 데는 일반적으로 밀리초가 걸리며, 복원하는 데는 데이터 볼륨에 따라 몇 초에서 몇 분이 걸립니다.</p>
<p>스냅샷 제한, 제한 사항 및 시스템에 미치는 영향에 대한 자세한 내용은 <a href="/docs/ko/snapshots.md">스냅샷을</a> 참조하세요.</p>
