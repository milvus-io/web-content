<p>Milvus Backup 便於在不同 Milvus 實例之間備份與還原元資料、區段及資料。它提供北向介面（例如 CLI、API 以及基於 gRPC 的 Go 模組），以便靈活操控備份與還原流程。</p>
<p>Milvus Backup 會從來源 Milvus 實例讀取集合元資料和分段，以建立備份。接著，它會將集合資料從來源 Milvus 實例的根目錄路徑複製出來，並將複製的資料儲存至備份根目錄路徑中。</p>
<p>若要從備份還原資料，Milvus Backup 會根據備份中的集合元資料和分段資訊，在目標 Milvus 實例中建立新的集合。接著，它會將備份資料從備份根目錄複製到目標實例的根目錄。</p>
