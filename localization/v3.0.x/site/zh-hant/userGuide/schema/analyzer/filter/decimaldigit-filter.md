<p>若要在自訂分析器中使用<code translate="no">decimaldigit</code> 過濾器，請將其新增至<code translate="no">analyzer_params</code> 中的<code translate="no">filter</code> 區段：</p>
<pre><code translate="no" class="language-python">analyzer_params = {
    <span class="hljs-string">&quot;tokenizer&quot;</span>: <span class="hljs-string">&quot;standard&quot;</span>,
    <span class="hljs-string">&quot;filter&quot;</span>: [<span class="hljs-string">&quot;decimaldigit&quot;</span>],
}
<button class="copy-code-btn"></button></code></pre>
<p><code translate="no">decimaldigit</code> 濾波器沒有可配置的參數。</p>
<p>此濾波器會將 Unicode 十進位數字（包括阿拉伯-印度數字、泰文數字、天城文數字、孟加拉文數字及全寬數字）轉換為 ASCII 數字。它針對詞性分析器所產生的詞元進行處理。上述配置是刻意作為自訂分析器的範例，並不包含完整的阿拉伯文處理管線。</p>
