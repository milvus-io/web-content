    </button></h1><p>Milvus ermöglicht es Ihnen, Standardwerte für skalare Felder (mit Ausnahme des Primärfeldes) festzulegen. Wenn für ein Feld ein Standardwert konfiguriert wurde, wendet Milvus diesen Wert automatisch an, wenn beim Einfügen keine Daten angegeben werden.</p>
<p>Standardwerte vereinfachen die Datenmigration von anderen Datenbanksystemen zu Milvus, da vorhandene Standardwerteinstellungen beibehalten werden. Sie können auch Standardwerte für Felder verwenden, deren Werte zum Zeitpunkt des Einfügens unsicher sein könnten.</p>

<li><p><code translate="no">JSON</code> und <code translate="no">ARRAY</code> Felder unterstützen keine Standardwerte.</p></li>
<li><p>Standardwerte können nur während der Erstellung der Sammlung konfiguriert und nicht nachträglich geändert werden.</p></li>
</ul>

<table>
   <tr>
     <th><p>Nullbar</p></th>
     <th><p>Standardwert</p></th>
     <th><p>Benutzereingabe</p></th>
     <th><p>Ergebnis</p></th>
   </tr>
   <tr>
     <td><p>✅</p></td>
     <td><p>✅ (Nicht-NULL)</p></td>
     <td><p>NULL oder weggelassen</p></td>
     <td><p>Verwendet den Standardwert</p></td>
   </tr>
   <tr>
     <td><p>✅</p></td>
     <td><p>❌</p></td>
     <td><p>NULL oder weggelassen</p></td>
     <td><p>Wird als NULL gespeichert</p></td>
   </tr>
   <tr>
     <td><p>❌</p></td>
     <td><p>✅ (Nicht-NULL)</p></td>
     <td><p>NULL oder weggelassen</p></td>
     <td><p>Verwendet den Standardwert</p></td>
   </tr>
   <tr>
     <td><p>❌</p></td>
     <td><p>❌</p></td>
     <td><p>NULL oder weggelassen</p></td>
     <td><p>Wirft einen Fehler</p></td>
   </tr>
   <tr>
     <td><p>❌</p></td>
     <td><p>✅ (NULL)</p></td>
     <td><p>NULL oder weggelassen</p></td>
     <td><p>Führt zu einem Fehler</p></td>
   </tr>

<p><strong>Wichtigste Erkenntnisse:</strong></p>
<ul>
<li><p>Wenn ein Feld einen Nicht-NULL-Standardwert hat, wird dieser Wert unabhängig davon verwendet, ob <code translate="no">nullable</code> aktiviert ist.</p></li>
<li><p>Wenn <code translate="no">nullable=True</code>, aber kein Standardwert festgelegt ist, speichert das Feld NULL.</p></li>
<li><p>Wenn <code translate="no">nullable=False</code> und kein Standardwert eingestellt ist, schlägt das Einfügen mit einem Fehler fehl.</p></li>
<li><p>Das Festlegen eines NULL-Standardwerts für ein nicht-nullbares Feld ist ungültig und verursacht einen Fehler.</p></li>
</ul>
