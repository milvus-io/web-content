    </button></h2><div class="alert note">
<p>对于阿拉伯语文本，在大多数情况下请使用内置的 <a href="/docs/zh/arabic-analyzer.md"><code translate="no">arabic</code></a> 分析器。该内置分析器集成了此过滤器，同时还包含标准分词、转小写、小数位规范化、阿拉伯语词干提取以及阿拉伯语停用词移除功能。仅当需要构建自定义分析器管道时，才应直接使用<code translate="no">arabic_normalization</code> 。</p>
</div>
<p>要在自定义分析器中使用<code translate="no">arabic_normalization</code> 过滤器，请将其添加到<code translate="no">analyzer_params</code> 中的<code translate="no">filter</code> 部分：</p>
<pre><code translate="no" class="language-python">analyzer_params = {
    <span class="hljs-string">&quot;tokenizer&quot;</span>: <span class="hljs-string">&quot;standard&quot;</span>,
    <span class="hljs-string">&quot;filter&quot;</span>: [<span class="hljs-string">&quot;arabic_normalization&quot;</span>],
}
<button class="copy-code-btn"></button></code></pre>
<p><code translate="no">arabic_normalization</code> 过滤器没有可配置的参数。</p>
<p>该过滤器执行以下转换：</p>
<table>
   <tr>
     <th><p>转换</p></th>
     <th><p>源</p></th>
     <th><p>转换为</p></th>

   <tr>
     <td><p>哈姆扎 + 阿莱夫变体</p></td>
     <td><p><code translate="no">آ</code>,<code translate="no">أ</code>,<code translate="no">إ</code></p></td>
     <td><p><code translate="no">ا</code></p></td>
   </tr>
   <tr>
     <td><p>带点“Teh”</p></td>
     <td><p><code translate="no">ة</code></p></td>
     <td><p><code translate="no">ه</code></p></td>
   </tr>
   <tr>
     <td><p>阿莱夫·马克苏拉</p></td>
     <td><p><code translate="no">ى</code></p></td>
     <td><p><code translate="no">ي</code></p></td>
   </tr>
   <tr>
     <td><p>哈拉卡特</p></td>
     <td><p><code translate="no">U+064B</code> 通过<code translate="no">U+065F</code></p></td>
     <td><p>已删除</p></td>
   </tr>
   <tr>
     <td><p>塔特维尔 / 卡希达</p></td>
     <td><p><code translate="no">ـ</code></p></td>
     <td><p>已移除</p></td>
   </tr>
</table>
