    </button></h1><p>本教程演示了如何构建一个同时搜索<strong>公共网络</strong>（通过<a href="https://exa.ai/">Exa</a>）和<strong>私人知识库</strong>（通过<a href="https://milvus.io/">Milvus</a>），然后合成统一答案的 Agents。该代理使用 OpenAI 的函数调用功能，根据用户的问题自动决定查询哪个来源。</p>
<p><a href="https://exa.ai/">Exa</a>是专为人工智能应用设计的搜索 API，它由<a href="https://zilliz.com/cloud">Zilliz Cloud</a>（全面管理 Milvus）提供技术支持，这也是<a href="https://zilliz.com/cloud">Zilliz Cloud</a>的骄傲。与传统的基于关键字的搜索引擎不同，Exa 支持语义（神经）搜索--您用自然语言描述您想要什么，它就能理解您的意图。它还提供内容提取、高亮显示和基于类别的过滤功能。<a href="https://milvus.io/">Milvus</a>是为可扩展的相似性搜索而构建的开源向量数据库。通过将它们与 LLM Agents 相结合，您可以在一个工作流中建立一个既能检索内部专有数据又能检索最新网络信息的系统。</p>
