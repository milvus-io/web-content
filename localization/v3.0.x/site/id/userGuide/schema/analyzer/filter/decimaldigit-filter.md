<p>Filter ` <code translate="no">decimaldigit</code> ` tidak memiliki parameter yang dapat dikonfigurasi.</p>
<p>Filter ini mengubah digit desimal Unicode, termasuk digit Arab-India, Thailand, Devanagari, Bengali, dan digit lebar penuh, menjadi digit ASCII. Filter ini beroperasi pada token yang dihasilkan oleh tokenizer. Konfigurasi di atas sengaja dibuat sebagai contoh penganalisis khusus dan tidak mencakup pipa pemrosesan bahasa Arab secara lengkap.</p>
