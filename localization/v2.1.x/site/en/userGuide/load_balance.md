      </svg>
    </button></h1><p>This topic describes how to balance query load in Milvus.</p>
<p>Milvus supports automatic load balance by default. You can <a href="/docs/v2.1.x/configure-docker.md">configure</a> your Milvus to enable or disable <a href="/docs/v2.1.x/configure_querycoord.md#queryCoordautoBalance">automatic load balance</a>. By specifying <a href="/docs/v2.1.x/configure_querycoord.md#queryCoordbalanceIntervalSeconds"><code translate="no">queryCoord.balanceIntervalSeconds</code></a>, <a href="/docs/v2.1.x/configure_querycoord.md#queryCoordoverloadedMemoryThresholdPercentage"><code translate="no">queryCoord.overloadedMemoryThresholdPercentage</code></a>, and <a href="/docs/v2.1.x/configure_querycoord.md#queryCoordmemoryUsageMaxDifferencePercentage"><code translate="no">queryCoord.memoryUsageMaxDifferencePercentage</code></a>, you can change the thresholds that trigger the automatic load balance.</p>
<p>If automatic load balance is disabled, you can still balance the load manually.</p>
