summary: >-
  O filtro `alphanumonly` remove tokens que contenham caracteres não-ASCII,
  mantendo apenas termos alfanuméricos. Este filtro é útil para processar texto
  onde apenas letras e números básicos são relevantes, excluindo quaisquer
  caracteres especiais ou símbolos.

<p>O filtro <code translate="no">alphanumonly</code> funciona com os termos gerados pelo tokenizador, pelo que deve ser utilizado em combinação com um tokenizador.</p>
<p>Depois de definir <code translate="no">analyzer_params</code>, pode aplicá-los a um campo <code translate="no">VARCHAR</code> ao definir um esquema de coleção. Isto permite que o Milvus processe o texto nesse campo utilizando o analisador especificado para uma tokenização e filtragem eficientes. Para mais pormenores, consulte <a href="/docs/pt/analyzer-overview.md#Example-use">Exemplo de utilização</a>.</p>
