<p>假設我們有一個 Milvus 實例，<code translate="no">milvus_A</code> ，使用名為<code translate="no">bucket_A</code> 的資料桶儲存資料。在這個範例中，我們的目標是完成下列任務：</p>
<ol>
<li><p>為<code translate="no">bucket_A</code> 中的 collection coll 建立備份 (<code translate="no">my_backup</code>) 。</p></li>
<li><p>從備份還原，並將還原的資料集命名為<code translate="no">coll_bak</code> 。</p></li>
</ol>

    </button></h2><h3 id="Step-1-Prepare-configuration" class="common-anchor-header">步驟 1：準備配置</h3><p>到 milvus-backup 專案的目錄，建立一個名為<code translate="no">configs</code> 的目錄：</p>
<pre><code translate="no" class="language-shell">mkdir configs
cd configs
<button class="copy-code-btn"></button></code></pre>
<p>下載備份設定檔案 backup.yaml：</p>
<pre><code translate="no" class="language-shell">wget https://raw.githubusercontent.com/zilliztech/milvus-backup/main/configs/backup.yaml
<button class="copy-code-btn"></button></code></pre>
<p>檔案結構如下：</p>
<pre><code translate="no">├── configs
│   └── backup.yaml
├── milvus-backup
└── README.md
<button class="copy-code-btn"></button></code></pre>

<h3 id="Step-3-Create-backup" class="common-anchor-header">步驟 3：建立備份</h3><p>儲存 backup.yaml 後，建立一個名為<code translate="no">my_backup</code> 的備份：</p>
<pre><code translate="no" class="language-shell">./milvus-backup create -c coll -n my_backup
<button class="copy-code-btn"></button></code></pre>
<p>此指令在<code translate="no">milvus_A</code> 的物件儲存中建立備份<code translate="no">bucket_A/backup/my_backup</code> 。</p>

    </button></h2><p>建立備份後，您可以使用以下指令還原：</p>
<pre><code translate="no" class="language-shell">./milvus-backup restore -c coll -n my_backup -s _bak
<button class="copy-code-btn"></button></code></pre>
<p>此命令從備份還原，並在<code translate="no">milvus_A</code> 中建立一個名為 coll_bak 的新集合，資料儲存在<code translate="no">bucket_A/files/insert_log/[ID of new collection]</code> 中。</p>
