        ></path>
      </svg>
    </button></h2><p>您可以根據您的系統架構，從<a href="https://github.com/milvus-io/milvus/releases/tag/v2.6.4">Milvus 發佈頁面下載</a> RPM/DEB 套件。</p>
<ul>
<li>對於 x86_64/amd64，請下載<strong>milvus_2.6.0-1_amd64.deb</strong>或<strong>milvus_2.6.0-1_amd64.rpm</strong>套件。</li>
<li>對於 ARM64，請下載<strong>milvus_2.6.0-1_arm64.deb</strong>或<strong>milvus_2.6.0-1_arm64.rpm</strong>套件。</li>
</ul>
<p>以下指令假設您要在 x86_64/amd64 機器上執行 Milvus Standalone。</p>
<pre><code translate="no" class="language-shell">wget https://github.com/milvus-io/milvus/releases/download/v2.6.4/milvus_2.6.0-1_amd64.rpm -O milvus_2.6.0-1_amd64.rpm
<button class="copy-code-btn"></button></code></pre>

        aria-hidden="true"

        ></path>
      </svg>
    </button></h2><p>要安裝 RPM/DEB 套件，您可以使用系統的套件管理員。</p>
<p>對於以 RPM 為基礎的系統 (例如 CentOS、Fedora 和 RHEL)，請使用<code translate="no">yum</code> 指令安裝套件。</p>
<pre><code translate="no" class="language-shell">yum install -y ./milvus_2.6.0-1_amd64.rpm
rpm -qa| grep milvus
<button class="copy-code-btn"></button></code></pre>
<p>對於以 DEB 為基礎的系統 (例如 Ubuntu 和 Debian)，請使用<code translate="no">apt</code> 指令來安裝套件。</p>
<pre><code translate="no" class="language-shell">apt install -y  ./milvus_2.6.0-1_amd64.deb
dpkg -l | grep milvus
<button class="copy-code-btn"></button></code></pre>

    </button></h2><p>安裝完成後，Milvus 會安裝為 systemd 服務，可以使用下列指令啟動：</p>
<pre><code translate="no" class="language-shell">systemctl start milvus
<button class="copy-code-btn"></button></code></pre>
<p>您可以使用下列指令檢查 Milvus 服務的狀態：</p>
<pre><code translate="no" class="language-shell">systemctl status milvus
<button class="copy-code-btn"></button></code></pre>
<p>如果 Milvus 運行成功，您應該會看到以下輸出：</p>

 <span class="hljs-attr">Main PID:</span> <span class="hljs-number">1044122</span> <span class="hljs-string">(milvus)</span>
