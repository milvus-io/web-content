<p>Supongamos que tenemos una instancia de Milvus, <code translate="no">milvus_A</code>, que utiliza un bucket llamado<code translate="no">bucket_A</code> para el almacenamiento de datos. En este ejemplo, nuestro objetivo es completar las siguientes tareas:</p>
<ol>
<li><p>Crear una copia de seguridad (<code translate="no">my_backup</code>) de la colección coll en <code translate="no">bucket_A</code>.</p></li>
<li><p>Restaurar desde la copia de seguridad y nombrar la colección restaurada <code translate="no">coll_bak</code>.</p></li>
</ol>

    </button></h2><h3 id="Step-1-Prepare-configuration" class="common-anchor-header">Paso 1: Preparar la configuración</h3><p>Vaya al directorio del proyecto milvus-backup y cree un directorio llamado <code translate="no">configs</code>:</p>
<pre><code translate="no" class="language-shell">mkdir configs
cd configs
<button class="copy-code-btn"></button></code></pre>
<p>Descargue el archivo de configuración de copia de seguridad backup.yaml:</p>
<pre><code translate="no" class="language-shell">wget https://raw.githubusercontent.com/zilliztech/milvus-backup/main/configs/backup.yaml
<button class="copy-code-btn"></button></code></pre>
<p>La estructura del archivo es la siguiente</p>
<pre><code translate="no">├── configs
│   └── backup.yaml
├── milvus-backup
└── README.md
<button class="copy-code-btn"></button></code></pre>

<h3 id="Step-3-Create-backup" class="common-anchor-header">Paso 3: Crear copia de seguridad</h3><p>Una vez guardado backup.yaml, cree una copia de seguridad denominada <code translate="no">my_backup</code>:</p>
<pre><code translate="no" class="language-shell">./milvus-backup create -c coll -n my_backup
<button class="copy-code-btn"></button></code></pre>
<p>Este comando crea la copia de seguridad <code translate="no">bucket_A/backup/my_backup</code> en el almacenamiento de objetos de <code translate="no">milvus_A</code>.</p>

      </svg>
    </button></h2><p>Una vez creada la copia de seguridad, puede restaurar a partir de ella utilizando el siguiente comando:</p>
<pre><code translate="no" class="language-shell">./milvus-backup restore -c coll -n my_backup -s _bak
<button class="copy-code-btn"></button></code></pre>
<p>Este comando restaura desde la copia de seguridad y crea una nueva colección llamada coll_bak en <code translate="no">milvus_A</code>, con los datos almacenados en <code translate="no">bucket_A/files/insert_log/[ID of new collection]</code>.</p>
