<p>Milvus Backup 支持在不同 Milvus 实例之间对元数据、分段和数据进行备份与恢复。它提供了北向接口（如 CLI、API 以及基于 gRPC 的 Go 模块），以便灵活地操作备份和恢复流程。</p>
<p>Milvus Backup 会从源 Milvus 实例读取 Collection 元数据和分段数据以创建备份。随后，它会从源 Milvus 实例的根路径复制 Collection 数据，并将复制的数据保存到备份根路径中。</p>
<p>要从备份中恢复数据，Milvus Backup 会根据备份中的 Collection 元数据和分段信息，在目标 Milvus 实例中创建一个新的 Collection。随后，它将备份数据从备份根路径复制到目标实例的根路径。</p>
