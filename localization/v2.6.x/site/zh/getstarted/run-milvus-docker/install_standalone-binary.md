      </svg>
    </button></h2><p>您可以从<a href="https://github.com/milvus-io/milvus/releases/tag/v2.6.19">Milvus 发布页面</a>根据您的系统架构下载相应的 RPM/DEB 软件包。</p>
<ul>
<li>对于 x86_64/amd64，请下载<strong>milvus_2.6.19-1_amd64.deb</strong>或<strong>milvus_2.6.19-1_amd64.rpm</strong>软件包。</li>
<li>对于 ARM64，请下载<strong>milvus_2.6.19-1_arm64.deb</strong>或<strong>milvus_2.6.19-1_arm64.rpm</strong>软件包。</li>
</ul>
<p>以下命令假设您将在 x86_64/amd64 机器上运行 Milvus Standalone。</p>
<pre><code translate="no" class="language-shell">wget https://github.com/milvus-io/milvus/releases/download/v2.6.19/milvus_2.6.19-1_amd64.rpm -O milvus_2.6.19-1_amd64.rpm
<button class="copy-code-btn"></button></code></pre>
<h2 id="Install-the-RPMDEB-Package" class="common-anchor-header">安装 RPM/DEB 软件包<button data-href="#Install-the-RPMDEB-Package" class="anchor-icon" translate="no">

      </svg>
    </button></h2><p>要安装 RPM/DEB 软件包，您可以使用系统的软件包管理器。</p>
<p>对于基于 RPM 的系统（如 CentOS、Fedora 和 RHEL），请使用<code translate="no">yum</code> 命令安装该软件包。</p>
<pre><code translate="no" class="language-shell">yum install -y ./milvus_2.6.19-1_amd64.rpm
rpm -qa| grep milvus
<button class="copy-code-btn"></button></code></pre>
<p>对于基于 DEB 的系统（如 Ubuntu 和 Debian），请使用<code translate="no">apt</code> 命令安装软件包。</p>
<pre><code translate="no" class="language-shell">apt install -y  ./milvus_2.6.19-1_amd64.deb
dpkg -l | grep milvus
<button class="copy-code-btn"></button></code></pre>

<button class="copy-code-btn"></button></code></pre>
<p>您可以使用以下命令检查 Milvus 服务的状态：</p>
<pre><code translate="no" class="language-shell">systemctl status milvus
<button class="copy-code-btn"></button></code></pre>
