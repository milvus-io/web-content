<li>Google protobuf ist installiert. Sie können es mit dem Befehl <code translate="no">pip3 install protobuf==3.20.0</code> installieren.</li>
<li>grpcio-tools ist installiert. Sie können es mit dem Befehl <code translate="no">pip3 install grpcio-tools</code> installieren.</li>
</ul>
