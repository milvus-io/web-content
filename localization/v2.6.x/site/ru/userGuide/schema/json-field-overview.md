        ></path>
      </svg>
    </button></h2><p>Чаще всего путаница возникает из-за разницы между полем JSON и <a href="/docs/ru/enable-dynamic-field.md">динамическим полем</a>. Хотя оба они связаны с JSON, они служат разным целям.</p>
<p>В таблице ниже приведены основные различия между полем JSON и динамическим полем:</p>
<table>
   <tr>
     <th><p>Характеристика</p></th>
     <th><p>JSON-поле</p></th>
     <th><p>Динамическое поле</p></th>
   </tr>
   <tr>
     <td><p>Определение схемы</p></td>
     <td><p>Скалярное поле, которое должно быть явно объявлено в схеме коллекции с типом <code translate="no">DataType.JSON</code>.</p></td>
     <td><p>Скрытое JSON-поле (с именем <code translate="no">#meta</code>), которое автоматически сохраняет необъявленные поля.</p></td>
   </tr>
   <tr>
     <td><p>Случай использования</p></td>
     <td><p>Хранение структурированных данных, где схема известна и согласована.</p></td>
     <td><p>Хранение гибких, изменяющихся или полуструктурированных данных, которые не вписываются в фиксированную схему.</p></td>
   </tr>
   <tr>
     <td><p>Контроль</p></td>
     <td><p>Вы контролируете имя и структуру поля.</p></td>
     <td><p>Неопределенные поля управляются системой.</p></td>
   </tr>
   <tr>
     <td><p>Запрос</p></td>
     <td><p>Запрос с использованием имени поля или целевого ключа внутри JSON-поля: <code translate="no">metadata["key"]</code>.</p></td>
     <td><p>Запрос напрямую с использованием динамического ключа поля: <code translate="no">"dynamic_key"</code> или через <code translate="no">#meta</code>: <code translate="no">#meta["dynamic_key"]</code></p></td>
   </tr>
</table>

<h4 id="Filtering-with-JSON-path-syntax" class="common-anchor-header">Фильтрация с помощью синтаксиса пути JSON</h4><p>Чтобы запросить конкретный ключ, используйте скобочную нотацию для доступа к ключам JSON: <code translate="no">json_field_name[&quot;key&quot;]</code>. Для вложенных ключей соедините их в цепочку: <code translate="no">json_field_name[&quot;key1&quot;][&quot;key2&quot;]</code>.</p>
<p>Чтобы отфильтровать сущности, для которых <code translate="no">category</code> является <code translate="no">&quot;electronics&quot;</code>:</p>
<pre><code translate="no" class="language-python"><span class="hljs-comment"># Define filter expression</span>
<span class="hljs-built_in">filter</span> = <span class="hljs-string">&#x27;metadata[&quot;category&quot;] == &quot;electronics&quot;&#x27;</span>

client.search(
