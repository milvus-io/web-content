<p>O Milvus Backup facilita o backup e a restauração de metadados, segmentos e dados entre instâncias do Milvus. Fornece interfaces northbound, tais como CLI, API e o módulo Go baseado em gRPC, para uma manipulação flexível dos processos de backup e restauração.</p>
<p>O Milvus Backup lê os metadados e segmentos da coleção a partir da instância Milvus de origem para criar um backup. Em seguida, copia os dados da coleção a partir do caminho raiz da instância Milvus de origem e guarda os dados copiados no caminho raiz do backup.</p>
<p>Para restaurar a partir de um backup, o Milvus Backup cria uma nova coleção na instância Milvus de destino com base nos metadados da coleção e nas informações dos segmentos contidos no backup. Em seguida, copia os dados do backup do caminho raiz do backup para o caminho raiz da instância de destino.</p>
