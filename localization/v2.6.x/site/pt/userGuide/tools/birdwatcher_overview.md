    </button></h1><p>Milvus é uma base de dados vetorial sem estado que separa leituras e escritas e tem o etcd a desempenhar o papel de fonte única de estado. Todos os coordenadores têm de consultar o estado a partir do etcd antes de poderem efetuar quaisquer alterações ao mesmo. Quando os utilizadores precisam de verificar ou limpar o estado, precisam de uma ferramenta para comunicar com o etcd. É aqui que o Birdwatcher entra em cena.</p>
<p>O Birdwatcher é uma ferramenta de depuração para o Milvus. Utilizando-o para se ligar ao etcd, pode verificar o estado do seu sistema Milvus ou configurá-lo em tempo real.</p>
