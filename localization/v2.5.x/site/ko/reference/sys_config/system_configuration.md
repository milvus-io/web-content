<p>#maxMemSize는 사용 가능한 GPU 메모리 전체를 초기화합니다.</p>
<p>이 섹션의 각 매개변수에 대한 자세한 설명은 <a href="/docs/ko/v2.5.x/configure_gpu.md">GPU 관련 구성을</a> 참조하세요.</p>
