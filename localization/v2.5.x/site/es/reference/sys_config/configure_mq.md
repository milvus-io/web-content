    </button></h1><p>Milvus supports four MQ: rocksmq(based on RockDB), natsmq(embedded nats-server), Pulsar and Kafka.</p>
<p>You can change your mq by setting mq.type field.</p>
<p>If you donâ€™t set mq.type field as default, there is a note about enabling priority if we config multiple mq in this file.</p>
<ol>
<li><p>standalone(local) mode: rocksmq(default) &gt; natsmq &gt; Pulsar &gt; Kafka</p></li>
<li><p>cluster mode:  Pulsar(default) &gt; Kafka (rocksmq and natsmq is unsupported in cluster mode)</p></li>
</ol>
