        ></path>
      </svg>
    </button></h2><p><code translate="no">english</code> 分析器使用以下组件：</p>
<ul>
<li><p><strong>标记化器</strong>：使用<code translate="no">standard</code><a href="/docs/zh/standard-tokenizer.md"> 标记化器</a>将文本分割成离散的单词单位。</p></li>
<li><p><strong>过滤器</strong>：包括多个过滤器，用于全面处理文本：</p>
<ul>
<li><p><code translate="no">lowercase</code>:将所有标记转换为小写，从而实现不区分大小写的搜索。</p></li>
<li><p><code translate="no">stemmer</code>:将单词还原为词根形式，以支持更广泛的匹配（例如，"running "变为 "run"）。</p></li>
<li><p><code translate="no">stop_words</code>:删除常见的英文停止词，以便集中搜索文本中的关键词语。</p></li>
</ul></li>
</ul>
