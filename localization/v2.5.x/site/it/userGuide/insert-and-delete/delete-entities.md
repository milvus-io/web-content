L'esempio di codice sottostante utilizza l'operatore <strong>in</strong> per eliminare in blocco tutte le entità con il campo <strong>colore</strong> impostato sui valori <strong>rosso</strong> e <strong>viola</strong>. È possibile utilizzare anche altri operatori per costruire espressioni di filtro che soddisfino le proprie esigenze. Per ulteriori informazioni sulle espressioni di filtro, consultare <a href="/docs/it/filtering">Filtraggio</a>.</p>
