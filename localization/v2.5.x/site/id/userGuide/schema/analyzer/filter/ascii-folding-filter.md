    </button></h1><p>Filter <code translate="no">asciifolding</code> mengubah karakter di luar <a href="https://en.wikipedia.org/wiki/Basic_Latin_(Unicode_block)">blok Unicode Latin Dasar</a> (127 karakter ASCII pertama) menjadi ekuivalen ASCII. Misalnya, filter ini mengubah karakter seperti <code translate="no">í</code> menjadi <code translate="no">i</code>, sehingga pemrosesan teks menjadi lebih sederhana dan konsisten, terutama untuk konten multibahasa.</p>

<p>Filter <code translate="no">asciifolding</code> beroperasi berdasarkan ketentuan yang dihasilkan oleh tokenizer, jadi filter ini harus digunakan bersama dengan tokenizer. Untuk daftar tokenizer yang tersedia di Milvus, lihat <a href="/docs/id/standard-tokenizer.md">Tokenizer Standar</a> dan halaman-halaman terkait.</p>
<p>Setelah mendefinisikan <code translate="no">analyzer_params</code>, Anda dapat menerapkannya ke bidang <code translate="no">VARCHAR</code> ketika mendefinisikan skema koleksi. Hal ini memungkinkan Milvus untuk memproses teks dalam bidang tersebut menggunakan penganalisis yang ditentukan untuk tokenisasi dan pemfilteran yang efisien. Untuk detailnya, lihat <a href="/docs/id/analyzer-overview.md#Example-use">Contoh penggunaan</a>.</p>
