        ></path>
      </svg>
    </button></h2><p>Penganalisis <code translate="no">english</code> menggunakan komponen-komponen berikut:</p>
<ul>
<li><p><strong>Tokenizer</strong>: Menggunakan<a href="/docs/id/v2.5.x/standard-tokenizer.md"> tokenizer</a> <code translate="no">standard</code><a href="/docs/id/v2.5.x/standard-tokenizer.md"></a> untuk membagi teks menjadi unit-unit kata yang terpisah.</p></li>
<li><p><strong>Filter</strong>: Termasuk beberapa filter untuk pemrosesan teks yang komprehensif:</p>
<ul>
<li><p><code translate="no">lowercase</code>: Mengubah semua token menjadi huruf kecil, sehingga memungkinkan pencarian tanpa huruf besar-kecil.</p></li>
<li><p><code translate="no">stemmer</code>: Mengurangi kata menjadi bentuk dasar untuk mendukung pencocokan yang lebih luas (misalnya, "berlari" menjadi "menjalankan").</p></li>
<li><p><code translate="no">stop_words</code>: Menghilangkan kata henti bahasa Inggris yang umum untuk fokus pada istilah-istilah kunci dalam teks.</p></li>
</ul></li>
</ul>
