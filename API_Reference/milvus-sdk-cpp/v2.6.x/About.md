# About Milvus C++ SDK

Milvus C++ SDK is an open-source project and its source code is hosted on [GitHub](https://github.com/milvus-io/milvus-sdk-cpp).

## Compatibility

| Milvus version | Recommended SDK version |
|:-----:|:-----:|
| 2.6.x | v2.6.4  |
| 3.0.x | v3.0.0  |

Note: Milvus and C++ SDK are NOT compatible across major versions.

## Installation

### Install from source

See [Development Guide](https://github.com/milvus-io/milvus-sdk-cpp/blob/master/DEVELOPMENT.md) for details on how to compile and install from the source.
